<a name="m3"></a> 
## 入口index.php内容

```php
header('Content-Type:text/html;charset=utf-8');
error_reporting(0);

require dirname(__FILE__).'/../../wavephp/Wave.php';
$configfile = dirname(__FILE__).'/protected/config/main.php';
// 不能起名$config
$wave = new Wave($configfile);
$wave->run();
```
